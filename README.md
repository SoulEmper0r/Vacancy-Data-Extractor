# Vacancy Data Extractor

Tampermonkey скрипт для автоматического извлечения данных о вакансиях с сайта HH.ru.

## 📋 Описание

Скрипт автоматически собирает информацию о требованиях к вакансиям на сайте HH.ru и выводит её в удобном формате. Полезен для HR-специалистов, рекрутеров и аналитиков, которые работают с большими объёмами данных о вакансиях.

## ✨ Возможности

- 🔄 **Автоматическое выполнение** - скрипт запускается сразу при открытии страницы вакансии
- 📊 **Извлечение данных** - собирает информацию из блоков с требованиями к кандидатам
- 📋 **Копирование в буфер обмена** - быстрый экспорт данных одним кликом
- 🎨 **Удобный интерфейс** - всплывающее окно с результатами справа вверху страницы
- 💾 **Вывод в консоль** - данные также доступны в консоли браузера (F12)

## 📦 Установка

### Способ 1: Прямая установка (рекомендуемый)

1. Установите расширение [Tampermonkey](https://www.tampermonkey.net/) для вашего браузера
2. Перейдите по [ссылке на установку](https://raw.githubusercontent.com/SoulEmper0r/Vacancy-Data-Extractor/main/HH.ru%20Vacancy%20Data%20Extractor.user.js)
3. Нажмите кнопку "Установить"

### Способ 2: Ручная установка

1. Установите расширение Tampermonkey
2. Откройте панель управления Tampermonkey
3. Нажмите "Создать новый скрипт"
4. Скопируйте содержимое файла `.user.js` из этого репозитория
5. Сохраните скрипт (Ctrl+S)

## 🚀 Использование

1. Перейдите на страницу любой вакансии на сайте [HH.ru](https://hh.ru)
2. Скрипт автоматически выполнится и извлечёт данные
3. В правом верхнем углу появится окно с результатами
4. Используйте кнопку "Копировать" для экспорта данных
5. Или откройте консоль браузера (F12) для просмотра данных там

## 🔧 Технические детали

- **Совместимость**: HH.ru (вакансии)
- **Зависимости**: Требуется расширение Tampermonkey
- **Формат вывода**: Текст (каждый пункт с новой строки)

## 📝 Формат данных

Скрипт извлекает данные из элементов `<li data-qa>` и выводит их в виде списка, где каждый элемент находится на новой строке. Это удобно для:
- Копирования в Excel/Google Sheets
- Анализа требований к вакансиям
- Создания отчётов
- Сравнения нескольких вакансий

## 🐛 Решение проблем

**Скрипт не работает:**
- Убедитесь, что Tampermonkey включён
- Проверьте, что скрипт активирован для домена hh.ru
- Обновите страницу (F5)

**Данные не извлекаются:**
- Возможно, изменилась структура сайта HH.ru
- Проверьте консоль браузера на наличие ошибок
